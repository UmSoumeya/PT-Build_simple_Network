# PT-Build_simple_Network

## What
Simulation réseau réalisée avec Cisco Packet Tracer représentant un réseau simple avec des hôtes, un switch et un routeur, permettant de tester la connectivité IP.

## Why
Ce projet a pour objectif de comprendre les bases du réseau :
- adressage IP
- passerelle par défaut
- communication entre machines
- validation de la connectivité (ping)

## When
Projet réalisé dans le cadre du bootcamp/Network and services.

## How
### Prérequis
- Cisco Packet Tracer

### Utilisation
1. Télécharger le fichier `Build_simple_network.pkt`
2. Ouvrir le fichier avec Cisco Packet Tracer
3. Vérifier la configuration IP des machines
4. Tester la connectivité avec des ping
